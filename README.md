# Digital Forensics & Incident Response Lab Portfolio

## Overview
This repository serves as a technical log of my practical studies in Digital Forensics and Incident Response (DFIR). The project documents my transition from Web Development to IT Security Engineering, showcasing the application of forensic tools and investigative techniques.

## Structure
Each directory in this repository represents a standalone laboratory exercise or a specific investigative case. Each lab contains:
* Technical documentation of the workflow.
* Findings and generated reports.
* Visual artifacts and methodology descriptions.

## Technical Scope
* **Evidence Acquisition:** Logical and physical imaging of various media.
* **Environment:** Virtualized lab environments (Kali Linux, Android Studio).
* **Forensic Toolset:** Industry-standard utilities for data extraction and analysis.
* **Documentation:** Structured reporting and chain of custody maintenance.
